# Resumen del producto

`Productive K3S Core` es la forma más simple de empezar con una base real de Kubernetes dentro del ecosistema Productive K3S.

Podés usarlo directamente cuando querés el camino más claro hacia la instalación base, la validación y la ejecución de addons o stacks curados.

También es la base sobre la que construye el resto del ecosistema:

- `productive-k3s-infra` monta la orquestación de despliegues sobre él
- `productive-k3s-cli` es la interfaz unificada recomendada por encima de él

El camino por defecto empieza con `k3s`, mientras que los usuarios más avanzados también pueden entender cómo ese mismo contrato crece hacia opciones como `rke2` o wiring alternativo de engines.
