# Resumen del producto

`Productive K3S` es una herramienta para bootstrapear, validar y operar un stack `k3s` orientado a producción sobre un host o una VM soportados.

En las páginas de abajo podés ver cómo se usa, por qué existe y qué plataformas están soportadas.
